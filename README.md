# Samuele Scaglioni — Personal Website

This repository hosts my personal academic website, built with HTML and CSS and published through GitHub Pages.

Website: [https://samuelescaglioni.github.io](https://samuelescaglioni.github.io)

## About

The website includes information about my academic background, research interests, selected projects, CV, and contact details.

I am an incoming MAE student at the University of Chicago. Before joining UChicago, I spent two years at the European Central Bank, working on macroprudential policy, financial stability, stress testing, and forecasting tools.

## Website structure

- `index.html` — homepage
- `research.html` — research and working papers
- `cv.html` — CV page
- `projects.html` — selected projects
- `contact.html` — contact information
- `styles.css` — website styling
- `images/` — profile and website images
- `files/` — CV and paper PDFs

## Notes

Professional project descriptions are kept at a high level and do not include confidential or internal material.
